Logo Usage Rules

DO

  • Use the SVG file at any size
  • Maintain clear space equal to the compass icon width on all sides
  • Use the dark version on dark/coloured backgrounds
  • Reproduce in full colour whenever possible

DON'T

  • Stretch, distort, or rotate the logo
  • Change the brand colours
  • Add drop shadows, outlines, or effects
  • Place on busy backgrounds without sufficient contrast
